🔌 IoT Ammonia Detection System

Complete Educational Circuit Diagram - Grade 7 Level

RED = Power (+5V) Supplies electricity
BLACK = Ground (GND) Return path
GREEN = Signals Arduino digital pins
YELLOW = Data Serial communication
CYAN = I²C I2C communication
ORANGE = Audio Speaker connections
📋 Complete Connection Guide (20 Wire Connections)

POWER CONNECTIONS:
  • ① Arduino 5V → Breadboard +5V Rail (provides power to all sensors)
  • ② Arduino GND → Breadboard GND Rail (completes the circuit)
  • ③ MQ-137 VCC → Breadboard +5V (powers ammonia sensor)
  • ④ MQ-137 GND → Breadboard GND (ammonia sensor return)
  • ⑥ LCD VCC → Breadboard +5V (powers display)
  • ⑦ LCD GND → Breadboard GND (display return)
  • ⑨ DFPlayer VCC → Breadboard +5V (powers audio module)
  • ⑩ DFPlayer GND → Breadboard GND (audio return)
  • ⑬ Relay VCC → Breadboard +5V (powers relay)
  • ⑭ Relay GND → Breadboard GND (relay return)
SENSOR & DATA CONNECTIONS:
  • ⑤ MQ-137 AO (Analog Out) → Arduino A0 (reads ammonia level)
  • ⑧ LCD I²C → Arduino A4/A5 (displays sensor readings)
  • ⑪ DFPlayer TX → Arduino D10 (receives audio commands)
  • ⑫ DFPlayer RX → Arduino D11 (sends audio data)
  • ⑮ Relay IN → Arduino D7 (controls mist maker)
OUTPUT CONNECTIONS:
  • ⑯ Arduino D5 → Green LED (shows normal operation)
  • ⑰ Arduino D6 → Red LED (shows high ammonia alert)
  • ⑱ DFPlayer → Speaker (plays alert sounds)
  • ⑲ Relay NO → Mist Maker (activates ultrasonic mist)
  • ⑳ Mist Maker → Breadboard GND (completes mist circuit)
