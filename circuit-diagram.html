System Overview:
This IoT Ammonia Detection System uses an Arduino Uno R3 as the main controller connected to:
  • MQ-137 Ammonia Sensor - Analog input (A0) measures ammonia concentration
  • 16×2 LCD Display - I²C communication (A4/A5) shows real-time readings
  • DFPlayer Mini - Audio alerts via serial communication (D10/D11)
  • Relay Module - Controls ultrasonic mist maker activation via D7
  • LED Indicators - Green (D5) = Normal, Red (D6) = High Ammonia
  • Power Distribution - USB power bank supplies all components via 5V rail
